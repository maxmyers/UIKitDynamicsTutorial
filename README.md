# Basic Animations

```

##### 2. Create UIDynamicAnimator Property

UIGravityBehavior *gravityBehavior = [[UIGravityBehavior alloc] initWithItems:@[WhatEverViewYouWant1,WhatEverViewYouWant2]];
```
